## HARDWARE
* É a parte física de um equipamento eletrônico
  - Entrada
  - Saída
  - Armazenamento
  - Processamento

## SOFTWARE
* Um conjunto de instruções que permite controlar um equipamento eletrônico.
* É a __parte lógica__ do computador.
  - Software de aplicação
  - Sofware de sistema

## Sistema Operacional
* Faz comunicação entre hardware e os demais softwares.
* O sistema Operacional cria uma plataforma comum a todos os programas utilizados.

## Número Binários
* ON = SIM = 1
* OFF = NÃO = 0
* A combinação de 0s e 1s em sequência define a informação que está sendo armazenada ou processada
* Tabela ASCII dicionário para o código binário. Cada caractere presente na tabela, como letras, números, símbolos e espaços, possui um código binário único associado a ele.
