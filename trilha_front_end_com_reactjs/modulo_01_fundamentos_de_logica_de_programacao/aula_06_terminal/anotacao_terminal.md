# Terminal e seus usos na programação
## Terminal
* Permite que conversemos com o computador de acordo com comandos (linhas escritas)
* é uma interface de linha de comando que permite aos usuários interagir com o sistema operacional por meio de textos, em vez de uma interface gráfica.
* Podemos executar comandos, gerenciar arquivos, rodar programas e realizar diversas tarefas adminitrativas.

## Saber caminho atual
```
$ pwd
```

## Listar arquivos
```
$ ls
```

## Entrar em diretório
```
$ cd nome_pasta
```


## Criar pasta via terminal
```
mkdir nome_pasta
```

## Voltar uma pasta
```
$ cd ..
```

## Criar arquivo
```
touch nome_arquivo
```

## Apagar pasta
```
$ rm -r nome_pasta
```

## Movendo arquivos
```
$ mv nome_arquivo pasta_destino
```

## Renomeando nomes
```
$ mv nome_arquivo_antigo nome_arquivo_novo
```
