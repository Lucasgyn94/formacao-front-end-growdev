# Representando Algoritmos - Pseudolinguagem
* Escreve-se em português (também chamado de __português estruturado__) sem ambiguidades e sem toda a rigidez de uma linguagem de programação.
* Importante notar que __um pseudo-código é escrito com frases que correspondem a estruturas utilizadas em uma linguagem de programção__
-> __Exemplo__
```
Algoritmo CalcularMediaAlunos
   LER Nota1
   LER Nota2
   LER Nota3
   media = (Nota1 + Nota2 + Nota3) / 3;
   SE media >= 6 ENTÃO
	IMPRIME "Aprovado"
   SENÃO
	IMPRIME "Reprovado"
FIM-SE
IMPRIME o valor da média
Fim-Algoritmo CalcularMediaAlunos
