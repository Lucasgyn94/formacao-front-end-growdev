# Vídeo
* https://www.youtube.com/watch?v=pdhqwbUWf4U

## Questões:
a) Qual é o principal intuito do pai nesse vídeo?
* O intuito do pai é ensinar às crianças o conceito básico de __algoritmos__ e ele faz isso assumindo o papel de uma pessoa totalmente sem noção de nada, como um robô, e segue as instruções como exatamente são dadas. O objetivo e demonstrar que para que uma tarefa seja realizada com sucesso, e necessário comandos precisos, claros e sequenciais, pois uma máquina faz apenas o que tiver explícito.

b) Qual é o principal problema que está ocorrendo entre o que as crianças estão escrevendo e o que o pai está fazendo?
* O principal problema é a forma das instruções das crianças em contraste com a interpretação do pai.

c) Aprofunde-se, pesquise:

O que é algoritmo e como isso pode ser exemplificado com o vídeo?
* Algoritmo é uma sequência de passos finitas e bem definidas para resolver um problema ou executar uma tarefa. No vídeo, algoritmo é o conjunto das instruções escritas pelas crianças no papel.

O que é input? Quais seriam os inputs nesse vídeo?
* Input são dados ou informações fornecidos ao sistema para serem processados.
* No vídeo os inputs seriam instruções como "espalhe a manteiga", "pegue a geleia", onde o pai recebe as frases como dados de entrada para saber o que fazer.

O que é processing? Quais seriam os processing nesse vídeo?
* Processing ou processamento é a etapa onde o computador executa as instruções do algoritmo sobre os dados de entrada (inputs) para transformá-los. É a "ação" propriamente dita.
* No vídeo processamento ou processing é o ato do pai ler a instrução e executar a ação física (enfiar a faca, enfiar no pote, passar no pão).

O que é output? Quais seriam os outputs nesse vídeo?
* Output é o resultado final obtido após o processamento. Nó vídeo é o estado final do sanduíche após cada tentativa.
