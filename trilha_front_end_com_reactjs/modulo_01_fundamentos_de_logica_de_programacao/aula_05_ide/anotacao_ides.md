# IDE - O que é? E as mais populares
## IDEs o que são?
* É um ambiente de desenvolvimento integrado
* São ferramentas utilizadas para escrever, testar e depurar nossos programas.

### Características Comuns de uma IDE
1. Editor de Código: Esta é a ferramenta principal de uma IDE, onde escrevemos nossos códigos. Os editores de códigos modernos oferecem destaque de sintaxe, autocompletar, e formatação automática para facilitar a leitura e escrita do código.

2. Compilador/Interpretador: A IDE geralmente inclui um compilador (para linguagens compiladas) ou um interpretador (para linguagens interpretadas) que converte o código-fonte escrito pelo programador em um formato que o computador pode executar.

3. Depurador: Uma ferramenta importante para encontrar e corrigir erros (bugs) no código. O depurador permite que os programadores executem seu código passo a passo, inspecionem variáveis e acompanhem a execução.

4. Gerenciador de Projetos: Ajuda a organizar os arquivos relacionados ao projeto, como código-fonte, bibliotecas, e outros recursos, tornando mais fácil navegar e administrar o projeto.

5. Integração com Sistemas de Controle de Versão: Muitas IDEs oferecem suporte para sistemas de controle de versão (como Git), permitindo que os programadores salvem diferentes versões do seu código e colaborem com outros desenvolvedores.

6. Ferramentas de Teste: Algumas IDEs incluem ferramentas para realizar testes automatizados, ajudando os programadores a garantir que seu código funcione corretamente.

7. Interfaces Gráficas: Muitas IDEs permitem a criação de interfaces gráficas do usuário (GUIs) de forma visual, facilitando o desenvolvimento de aplicativos com uma interface de usuário amigável.
