# GIT - Versionando os códigos
## Git
* __Git__: Sistema de versionamento (máquina do tempo do código). Roda localmente.

## Controle de Versão
* Permite que agente __reverta para estado anterior determinados arquivos ou um projeto inteiro.__

* __Se estragarmos tudo ou perder arquivos, conseguimos recuperar facilmente.__

## Comandos Essenciais

### Configurar as informações
```
$ git config --global user.name "[nome]"
$ git config --global user.email "[endereco-de-email]"
```

### Inicializar
```
$ git init [nome-do-projeto]
```

### Criar e acessar a branch
```
$ git checkout -b [nome-da-branch]
```

### Verificar o status da aplicação
```
$ git status
```

### Adiciona as informações
```
$ git add arquivo.extensao
```

### Commita as informações
```
$ git commit -m "Adiciona a mensagem"
```
