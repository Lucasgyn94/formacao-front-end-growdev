# Fixando...

* Siga os passos abaixo usando apenas o terminal (Git Bash) :

1. No seu desktop, crie um diretório chamado fixando_git
2. Inicialize o git nesse diretório
3. A partir da branch principal, crie e acesse uma nova branch de nome teste-inicial
4. Crie um arquivo chamado infos.txt e coloque a informações sobre você
5. Verifique o status e veja se foi alterado
6. Commite as alterações
7. Refaça esses processos com as informações de outra pessoa conhecida
