# Fixando...

* Siga os passos abaixo usando apenas o terminal (Git Bash) :

1. Crie o perfil no Github
2. Clone o diretório do seu perfil na sua máquina local
3. A partir da branch principal, crie e acesse uma nova branch de nome versionamento-inicial
4. Crie um arquivo chamado README.md e coloque informações sobre você (hobbies, formação, experiência
profissional, linguagem de programação, o que está estudando atualmente)
5. Verifique o status e veja se foi alterado
6. Commite as alterações
7. Abra um PR e mescle as alterações deixando o seu perfil completo
