## GitHub
* __GitHub__: Plataforma na nuvem para hospedar repositórios Git (rede social de dev).

### Clonar o repositório
```
$ git clone link_repo
```

### Criar e acessar a branch
```
$ git checkout -b [nome-da-branch]
```

### Verificar o status da aplicação
```
$ git status
```

### Adiciona as informações
```
$ git add arquivo.extensao
```

### Commita as informações
```
$ git commit -m "Adiciona a mensagem"
```

### Subir do Local para o Remoto
```
$ git push origin 'nome da branch' 
```

### Atualizar as branches
```
$ git pull origin 'nome da branch'
```
