# Elementos HTML, Headings, Parágrafos e Formatação
## Headings
* Organizam os textos em seções separados por tais tipos, separados por 6 niveís: h1, h2, h3, h4, h5, h6.

## Parágrafos
* Utilizados para inserção de textos

### Exemplo
<p> </p>

## Formatação
* São tags que dão estilos aos nossos textos

### Exemplo
<b>, <strong>, <i>, <em>, <mark>, <small>, <del>, <ins>, <sub>, <sup>
