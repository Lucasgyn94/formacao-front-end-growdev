Introdução ao HTML

O que é HTML ?

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar o conteúdo de páginas da web, como textos, imagens, links e vídeos, usando "tags" para organizar e dar significado a esses elementos para o navegador, funcionando como o esqueleto de qualquer site para definir o que é título, parágrafo ou botão. Serve para desenvolver a base de sites e aplicativos online, permitindo que o conteúdo seja exibido corretamente e que haja navegação entre diferentes páginas e recursos, sendo essencial para a internet como a conhecemos.

Principais Tags de Formatação

1. (Bold)

Esta tag serve para estilizar o texto em negrito (bold).

2. Strong

Esta tag define que o texto contido nela tem forte importância, seriedade ou urgência. Embora visualmente a maioria dos navegadores também a exiba em negrito (igual ao b ), semanticamente ela indica aos motores de busca e leitores de tela que aquela informação é crucial.

3. (Italic)

A tag i é usada para diferenciar uma parte do texto do restante por algum motivo específico, como termos técnicos, expressões em outro idioma, nomes de obras ou pensamentos. Visualmente, o texto fica em itálico, mas sem aplicar ênfase na entonação de leitura.

4. (Emphasis)

Esta tag serve para dar ênfase (emphasis) a uma palavra ou trecho. Diferente do i , a tag em altera o sentido da frase, indicando que aquela parte deve ser lida com uma entonação diferente ou estresse verbal. Visualmente, também é apresentada em itálico.

5. (Marked)

A tag mark é utilizada para destacar ou realçar uma parte do texto, geralmente para fins de referência ou notação, como se você tivesse passado um "marca-texto" amarelo sobre a palavra. É muito comum para indicar termos encontrados em uma busca.

6. (Small Print)

Esta tag é usada para representar "letras miúdas", como comentários laterais, avisos legais, direitos autorais (copyright) ou isenções de responsabilidade. O navegador renderiza o texto em um tamanho ligeiramente menor que o padrão.

7. (Deleted Text)

A tag del indica um texto que foi removido ou deletado de um documento. É frequentemente usada em listas de tarefas para marcar itens concluídos ou em lojas virtuais para mostrar o preço antigo de um produto. Visualmente, o texto aparece tachado (riscado).

8. (Subscript)

Esta tag define texto subscrito, ou seja, o texto aparece posicionado ligeiramente abaixo da linha normal e em tamanho menor. É extremamente útil para escrever fórmulas químicas.

9. (Superscript)

A tag sup define texto sobrescrito, onde o texto aparece posicionado ligeiramente acima da linha normal e em tamanho menor. É comumente utilizada para representar expoentes matemáticos, numerais ordinais ou referências de notas de rodapé.
